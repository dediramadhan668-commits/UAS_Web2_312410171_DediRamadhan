# 📖 Deskripsi

E-Report Pengaduan Masyarakat merupakan aplikasi berbasis web yang digunakan untuk membantu proses pengelolaan laporan pengaduan masyarakat.

Aplikasi ini dibangun menggunakan **CodeIgniter 4**, **PHP**, **MariaDB/MySQL**, dan **Bootstrap 5**.

# 🚀 Cara Menjalankan Project

Clone repository

```bash
git clone https://github.com/dediramadhan668-commits/UAS_Web2_312410171_DediRamadhan.git
```

Masuk ke folder project

```bash
cd UAS_Web2_312410171_DediRamadhan/backend-api
```

Install dependency

```bash
composer install
```

Jalankan server

```bash
php spark serve
```

Buka browser

```
http://localhost:8080
```
